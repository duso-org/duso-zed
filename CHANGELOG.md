# Changelog

## [0.1.0] - 2026-03-08

### Added
- Initial release of Duso extension for Zed
- Syntax highlighting for `.du` files
- Language Server Protocol (LSP) integration with `duso -lsp`
- Support for keywords, built-in functions, strings, and comments
- Bracket matching and auto-indentation
- Code outline support
- Helpful code snippets for common patterns:
  - Functions, control flow (if/else, for, while)
  - Try/catch blocks
  - Parallel execution
  - Module imports and Claude integration

### Features
- Full syntax highlighting for Duso language features
- Real-time error checking via LSP
- Auto-completion support (provided by LSP)
- Code navigation and outline
- Proper indentation handling for Duso blocks
